## ✨ What is Market Live?

Market Live is a zero-latency trading dashboard designed for speed, clarity, and aesthetics. It connects directly to the **Binance WebSocket API** to stream live crypto prices, while using advanced simulation logic for stocks, commodities, and forex. 

Built with **Next.js 15**, it’s a lightweight but powerful terminal that requires **zero setup** and **no API keys** to run. Just open it and start tracking.

## 🛠️ Tech Stack

*   **Framework**: [Next.js 15](https://nextjs.org/) (The latest & greatest)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/) + Custom Modern CSS
*   **Motion**: [Framer Motion](https://www.framer-motion.com/) (For that hyper-fluid feel)
*   **Charts**: [Lightweight Charts](https://tradingview.github.io/lightweight-charts/) (Pro trading engine)
*   **Icons**: [Lucide React](https://lucide.dev/) + High-quality SVG brand logos
*   **Language**: [TypeScript](https://www.typescriptlang.org/) (Strictly typed for reliability)

## 🚀 Quick Start

Ready to run your own terminal? It takes less than 2 minutes.

### 1. Clone & Install
```bash
git clone https://github.com/YOUR_USERNAME/market-live.git
cd market-live
npm install
```

### 2. Ignite the Dev Server
```bash
npm run dev
```
Head to `http://localhost:3000` — you're live! 🏎️💨
